# Линейный конгруэнтный генератор (LCG)

LCG - генератор детерминирован: одно и то же семя всегда даёт одну и ту же последовательность.

Отправитель и получатель с одним паролем получат одинаковый ключевой поток.

```
state = state &* 6364136223846793005 &+ 1442695040888963407
```
6364136223846793005 - Множитель Кнута

1442695040888963407 - Приращение Кнута

&* и &+ - арифметика с намеренным переполнением. mod 2^64

```
return UInt8((state >> 33) & 1)
```
Тут мы делаем сдвиг вправо на 33 мы отбрасываем 33 младших бита и берем 8 следующих

Чем выше бит - тем на большее кол-во предыдущих битов он опирается, тем сложнее его поведение и тем длиннее его период.

Поэтому мы первые 33 бита выбрасываем - они слишком предсказуемы.

Берем следующие 8 бит потому что они уже достаточно случайны.

# Потоковый XOR-шифр

Потоковый XOR-шифр берет каждый байт файла и смешивает его с байтом ключевого потока через операцию XOR.

Ключевой поток - бесконечная последовательность псевдослучайных байтов которую выдает генератор LCG.

XOR - это побитовая операция у которой есть свойство - если применить дважды с одним и тем же ключом, то получим его обратно.

По сути шифрование и дешифрование это одна и та же функция и не нужен другой алгоритм.

```
func xorStreamChunk(_ chunk: ArraySlice<UInt8>, prng: inout LCGGenerator) -> [UInt8]
```
Тут мы принимаем кусок файла - срез массива байтов

inout - ссылка, она необходима чтобы помнить свое состояние между вызовами, чтобы следующий блок продолжал поток, а не начинал с начала.

```
chunk.map { $0 ^ prng.nextByte() }
```
map проходит по каждому байту куска.

$0 - это текущий байт файла. 

prng.nextByte - следующий байт ключевого потока от генератора

Получается новый массив зашифрованных байтов того размера.

##  SHA-1

Сначала он получает исходное сообщение кратное 512 бит

Добавляется 1 бит а затем 0 в конце записывается длина исходного сообщения в 64 битах.

Это нужно чтобы алгоритм работал всегда с блоками одного размера

После создаются пять 32 битных переменных с фиксированным начальным значениям.

Каждый 512 битный блок проходит через 80 преобразований

После обработки всез блоков пять буферов соединяются в одну 160 битную строку - это и есть итог.
```
Insecure.SHA1.hash(data: data)
```
считываем хеш и возвращаем его в виде читаемой строки

так же на выходе получаем 20 байт

```
.map { String(format: "%02x", $0) }
.joined()
```

Каждый из 20 байтов превращаем в двухсимвольномю строку - %02x

Затем все 20 кусочков склеиваются в одну строку длиной 40 символов. 

```
let digest = Insecure.SHA1.hash(data: Data(password.utf8))
```

Сначала хешируем пароль. password.utf8 — преобразуем строку пароля в байты 
```
var seed: UInt64 = 0
for (i, byte) in digest.prefix(8).enumerated() {
    seed |= UInt64(byte) << (56 - i * 8)
}
```
Берём только первые 8 байт из 20.

Дальше собираем их в одно число первый байт встаёт на самое старшее место (сдвиг на 56 бит), второй — чуть правее (сдвиг на 48), и так далее до последнего байта без сдвига.

# ГОСТ (Режим гаммирования)

Блочный шифр обрабатывает блоками по 64 бита и ключ длиной 256 бит.

Для начала создаем таблицу замены. 32 битное слово делится на 8 частей по 4 бита

Без нее шифр был бы линейным.

```
struct GOST28147 {
    static let sBox: [[UInt8]] = [
        [9,  6,  3,  2,  8, 11,  1,  7, 10,  4, 14, 15, 12,  0, 13,  5],
        [3,  7, 14,  9,  8, 10,  2,  5,  6,  0, 11, 15, 13,  1,  4, 12],
        [14,  4,  6,  2, 11,  3, 13,  8, 12, 15,  5, 10,  0,  7,  1,  9],
        [14,  7, 10, 12, 13,  1,  3,  9,  0,  2, 11,  4, 15,  8,  5,  6],
        [11,  5,  1,  9,  8, 13, 15,  0, 14,  4,  2,  3, 12,  7, 10,  6],
        [ 3, 10, 13, 12,  1,  2,  0, 11,  7,  5,  9,  4,  8, 15, 14,  6],
        [ 1, 13,  2,  9,  7, 10,  6,  0,  8, 12,  4,  5, 15,  3, 11, 14],
        [11, 10, 15,  5,  0, 12, 14,  8,  6,  2,  3,  9,  1,  7, 13,  4]
    ]
```
Создаем шаг счетчика гаммирования и на каждом шаге он увеличивается.

Он гарантирует что счетчик не зациклится раньше времени.
```
static let delta1: UInt32 = 0x01010101
static let delta2: UInt32 = 0x01010104
```

В начале мы делаем проверку на 32 байта

Потом инициализируем цикл который выполняется 8 раз

b - смещение с которого начинается группа из 4 байт

После собираем 4 байта в единое число

Первый байт стал младшим а последний старшим.

На выходе получаем массив из 8 подключей
```
static func keySchedule(from keyBytes: [UInt8]) -> [UInt32] {
        precondition(keyBytes.count == 32, "ключ должен быть 32 байта")
        return (0..<8).map { i in
            let b = i * 4
            return UInt32(keyBytes[b])
                 | UInt32(keyBytes[b + 1]) << 8
                 | UInt32(keyBytes[b + 2]) << 16
                 | UInt32(keyBytes[b + 3]) << 24
        }
    }
```
Сначала производим сложение с подключом по модолю 2 32

Смешиваем входное слово с подключом текущего раунда и результат ищет замену

Переходим к циклу

32 битное режется на 8 кусков по 4 бита (нибболов)

после сдвигаем нужное число на позицию 0

В конце происходит циклический сдвиг влево на 11 бит

Сдвиг нужен для того, чтобы изменение одного бита после блоков повлияло на несколько нибблов в следующем раунде

По сути перемешали с ключом, произвели нелинейную замену и распредили биты по всему слову
```
 static func roundFunction(_ x: UInt32, subkey: UInt32) -> UInt32 {
        let sum = x &+ subkey
        var sub: UInt32 = 0
        for n in 0..<8 {
            let nibble = Int((sum >> (n * 4)) & 0xF)
            sub |= UInt32(sBox[n][nibble]) << (n * 4)
        }
        // Циклический сдвиг влево на 11 позиций
        return (sub << 11) | (sub >> 21)
    }
```

В конце переставляем тк они возвращаются в перестановленном проядке, тк сдели лишний обмен и приводим к правильному виду

for ki in schedule {

## Вывод ключа из пароля
Гост требует ровно 32 байта, а sha-1 только 20. Поэтому хешируем пароль дважды с разными данными h1 и h2.

deriveIV получаем 8 байтную синхронпосылку из пароля

хешируем строку + пароль и берем первые 8 байт

по сути три разных хеша от одного пароля и все три дают разные байты.
```
 static func deriveKey(from password: String) -> [UInt8] {
        let base = Data(password.utf8)
        let h1   = Array(Insecure.SHA1.hash(data: base))
        let h2   = Array(Insecure.SHA1.hash(data: base + Data("key2".utf8)))
        return Array((h1 + h2).prefix(32))
    }

    static func deriveIV(from password: String) -> [UInt8] {
        Array(Insecure.SHA1.hash(data: Data(("gost_iv_" + password).utf8)).prefix(8))
    }
}
```
