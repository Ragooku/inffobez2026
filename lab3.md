# Линейный конгруэнтный генератор (LCG)

LCG - генератор детерминирован: одно и то же семя всегда даёт одну и ту же последовательность.

Отправитель и получатель с одним паролем получат одинаковый ключевой поток.

```
state = state &* 6364136223846793005 &+ 1442695040888963407
```
6364136223846793005 - Множитель Кнута

1442695040888963407 - Приращение Кнута

&* и &+ - арифметика с намеренным переполнением. mod 2^64

```
return UInt8((state >> 33) & 1)
```
Тут мы делаем сдвиг вправо на 33 мы отбрасываем 33 младших бита и берем 8 следующих

Чем выше бит - тем на большее кол-во предыдущих битов он опирается, тем сложнее его поведение и тем длиннее его период.

Поэтому мы первые 33 бита выбрасываем - они слишком предсказуемы.

Берем следующие 8 бит потому что они уже достаточно случайны.

# Потоковый XOR-шифр
Данная функция вычисляет побитовое исключающее ИЛИ. Через логические операции.

(a | b) & ~(a & b) - булевое выражение

a | b — биты, где хотя бы один равен 1

a & b — биты, которые одинаковые и равны 1

(a | b) & ~(a & b) — оставляет только те биты, где ровно один из операндов равен 1

```
func xorBytes(_ a: UInt8, _ b: UInt8) -> UInt8 {
    return (a | b) & ~(a & b)
}
```

Создаётся пустой массив result для хранения результата.

Каждый новый чанк продолжает генерацию гаммы с того места, где остановился предыдущий — поэтому поток остаётся непрерывным.

Функция проходит по каждому байту переданного куска данных и побитово складывает его (через XOR) с очередным байтом ключевого потока.

В результате возвращается новый массив байтов — уже зашифрованный (или расшифрованный).

```
func xorStreamChunk(_ chunk: ArraySlice<UInt8>, prng: inout LCGGenerator) -> [UInt8] {
    var result = [UInt8]()
    result.reserveCapacity(chunk.count)
    for byte in chunk {
        result.append(xorBytes(byte, prng.nextByte()))
    }
    return result
}
```

##  SHA-1

Сначала он получает исходное сообщение кратное 512 бит

Добавляется 1 бит а затем 0 в конце записывается длина исходного сообщения в 64 битах.

Это нужно чтобы алгоритм работал всегда с блоками одного размера

После создаются пять 32 битных переменных с фиксированным начальным значениям.

Каждый 512 битный блок проходит через 80 преобразований

После обработки всез блоков пять буферов соединяются в одну 160 битную строку - это и есть итог.
```
Insecure.SHA1.hash(data: data)
```
считываем хеш и возвращаем его в виде читаемой строки

так же на выходе получаем 20 байт

.map { String(format: "%02x", $0) }

Каждый из 20 байтов превращаем в двухсимвольномю строку - %02x

Затем все 20 кусочков склеиваются в одну строку длиной 40 символов. 

```
let digest = Insecure.SHA1.hash(data: Data(password.utf8))
```

Сначала хешируем пароль. password.utf8 — преобразуем строку пароля в байты 
```
var seed: UInt64 = 0
for (i, byte) in digest.prefix(8).enumerated() {
    seed |= UInt64(byte) << (56 - i * 8)
}
```
Берём только первые 8 байт из 20.

Дальше собираем их в одно число первый байт встаёт на самое старшее место (сдвиг на 56 бит), второй — чуть правее (сдвиг на 48), и так далее до последнего байта без сдвига.
